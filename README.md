## 상세 설명

### `cart-domain`

- **설명**: 장바구니 예시를 통한 비즈니스 로직 분리 및 테스트코드 작성법
  - 비즈니스 로직과 뷰 로직을 적절하게 분리할 수 있다.
  - 분리된 비즈니스 로직은 한번의 테스트로도 견고하고 완성도 있는 프로젝트를 제공한다.
